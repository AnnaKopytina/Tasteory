# Техническое задание Tasteory

### Проект:
Веб-приложение для хранения кулинарных рецептов внутри групп и не только и обмена ими с другими людьми.

### Цель проекта:
Создать веб-приложение, которое позволит людям хранить, организовывать кулинарные рецепты, решая проблему их разрозненного хранения, потери и долгого поиска, благодаря удобному интерфейсу с возможностью добавлять фото, видео и различные заметки, а также создавать закрытые группы для обмена рецептами с выбранными людьми

### Описание проекта:
Tasteory предназначен для создания единого цифрового хранилища рецептов. Пользователи смогут добавлять рецепты, редактировать их, структурировать по категориям, быстро находить нужные блюда и делиться ими с другими людьми внутри определенного круга пользователей (например, друзей, семьи или небольшого сообщества) и не только.

### Требования к интерфейсу:
Главный экран приложения должен содержать ленту из общедоступных рецептов, строку поиска по рецептам, фильтрацию рецептов, список групп пользователя и кнопку для добавления нового рецепта.
Пользователь должен иметь возможность создавать закрытые группы, приглашать других пользователей в группу, публиковать рецепты внутри группы.
Должна быть возможность добавления нового рецепта с указанием названия, ингредиентов, пошагового описания приготовления, прикрепления фото, видео, редактирования и удаления рецептов, просмотра подробной страницы рецепта.
Приложение должно поддерживать аутентификацию и разделять роли в группе: «Владелец» и «Участник».
Вступление в семейную группу должно происходить по уникальному коду приглашения.
Сделать превью рецепта и страницу рецепта с полной информацией.

### Дополнительные требования:
Приложение должно поддерживать динамический пересчет количества продуктов в зависимости от выбранного числа порций.
Для каждого этапа приготовления должна быть предусмотрена возможность добавления общегрупповой заметки.
